# Benco Box

Professional landing page for Benco Box, an eco-friendly meal prep company.

## Files

- `index.html` — main website page
- `css/styles.css` — professional responsive styling
- `js/script.js` — simple interactive behavior
- `assets/benco-box-logo.png` — brand logo

## How to view

Open `index.html` in a browser, or publish this repo with GitHub Pages.
